# openblock-driver

This repo does not contain any source code. It is just used to release binary files of drivers.

## Support system

- Windows x86/x64
- MacOS
- Linux

## Windows

**Driver list**

- Arduino dpinst
- CH341SER
- CP210x
- FTDI USB Drivers
- mbedWinSerial

## MacOS

**Driver list**

- CH341SER
- CP210x
- FTDI USB Drivers

## Linux

**Driver list**

- Null. Contains only a script to configure serial port permissions.
